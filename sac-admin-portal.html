dns லைவ் டேட்டாபேஸ் இணைப்பு (Firebase)

இணையதளத்தை கோப்பு திருத்தாமல், லைவ் டேட்டாபேஸில் இணைக்க உங்கள் Firebase Firestore விபரங்களை பதிவிடவும். இயக்கப்பட்டவுடன், இந்த வலைத்தளம் நேரடியாக உங்கள் Firebase இல் தகவல்களை சேமிக்கும்.

domain டொமைன் மற்றும் ஹோஸ்டிங் வழிகாட்டி

1. Free Hosting (Vercel & Netlify)

இந்த இணையதளம் தூய HTML, CSS, JS கொண்டுள்ளதால் Vercel, Netlify அல்லது GitHub Pages இல் முற்றிலும் இலவசமாக ஹோஸ்ட் செய்யலாம்.

2. Connecting Custom Domain

ஹோஸ்டிங் தளத்தில் உங்கள் டொமைனை (e.g. www.stantonyschurchpaganur.in) இணைக்க, டொமைன் வாங்கிய தளத்தில் (GoDaddy, Hostinger) DNS CNAME அல்லது A Record ஐ மாற்ற வேண்டும்.

3. Git Repository Branches

உங்களது ரெபாசிட்டரியில் DEV மற்றும் Main என இரு பிரான்சுகள் உருவாக்கப்பட்டுள்ளன. DEV பிரான்சில் புதிய மாற்றங்களை செய்து சோதித்துவிட்டு, Main பிரான்சில் மெர்ஜ் செய்யும் போது அது உடனடியாக இணையதளத்தில் அப்டேட் ஆகும் (CI/CD auto-deploy).
